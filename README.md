# DPDR Symptom Management Web

A website project focused on symptom management resources, self-monitoring, psychoeducation, and supportive tools for depersonalization-derealization disorder (DPDR).

## Initial vision
- Symptom tracking
- Trigger and coping log
- Psychoeducation pages
- Grounding toolkit
- Crisis/support resources
- Privacy-first design
